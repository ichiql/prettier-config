# @ichiql/prettier-config

## install

```shell
pnpm add @ichiql/prettier-config
npm install @ichiql/prettier-config
yarn add @ichiql/prettier-config
```

## config file for prettier

### .eslintrc.cjs

add `prettier` to .eslint.cjs

### package.json

```json:package.json
{
  ~~~
  "prettier": "@ichiql/prettier-config"
  ~~~
}
```
